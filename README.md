# Academic Management System

## Overview

The Academic Management System is a SQL-based database project designed to manage students, courses, instructors, and enrollments within an educational institution.

This project demonstrates the use of relational database concepts including table creation, primary keys, foreign keys, data manipulation, and SQL JOIN operations.

---

## Features

* Student Management
* Course Management
* Instructor Management
* Enrollment Tracking
* Course-Instructor Mapping
* Relational Database Design
* SQL JOIN Queries

---

## Database Tables

### Students

Stores student information such as ID, name, email, and admission date.

### Courses

Stores available course details and credits.

### Instructors

Stores instructor information and departments.

### Enrollments

Tracks course registrations made by students.

### Course Assignments

Maps instructors to the courses they teach.
